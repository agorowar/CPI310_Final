Project Plan

An outline of our anticipated dates of feature completion and milestones as well as features that may be added or lost depending on time constraints.


Schedule

  • 10/21 - Website and Pitch
  • 10/25 - UI design Mockup.
  • 10/28 - UI wireframe
  • 10/31 - Create pet profile pages for users to log information about their pets
  • 11/5 - Pre-prototype check-in
  • 11/7 - Create owner profile pages
  • 11/14 - Set up a system that shows users pets so that they can match with other pets
  • 11/16 - Prototype
  • 11/21 - Create user database so customers can create accounts for their pets
  • 11/28 - Create a special page for adoption of various pets
  • 12/5 - Integration of back-end and front-end

Features that may be added if time permits:

  • Live chatting between users
  • Maps displaying location of potential adoptions
  • Maps displaying dog parks in the area

Features that may be lost if time limits:

  • Create a special page for adoption of various pets
  • Support for pets outside cats and dogs
